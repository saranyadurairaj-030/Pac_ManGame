## Project Description
This project is a Java-based implementation of the classic Pac-Man game using Swing and AWT. The game allows the player to control Pac-Man using keyboard keys, collect food, 
avoid ghosts, and progress through multiple levels.

## Technologies Used
- Java
- Swing & AWT
- Eclipse IDE

## Features
- Multiple levels
- Four ghosts
- Score calculation
- Game over and retry option
